1. Mengubah Waktu menjadi Laju (Rate)

Kita perlu tahu berapa pelanggan yang datang per menit.

  • λ (Laju Kedatangan) = 1 ÷ {{ results.input_arrival }}
    → λ = {{ results.lam }} pelanggan/menit
  • μ (Laju Pelayanan) = 1 ÷ {{ results.input_service }}
    → μ = {{ results.mu }} pelanggan/menit
2. Menghitung Pemanfaatan Server (ρ)

Mengukur seberapa sibuk kedua pelayan.

Rumus: ρ = λ ÷ (2 × μ)

Substitusi: ρ = {{ results.lam }} ÷ (2 × {{ results.mu }})
Hitung: ρ = {{ results.lam }} ÷ {{ results.calc_2mu }}
Hasil: ρ = {{ results.rho }} (atau {{ results.rho_percent }}%)
3. Waktu Rata-rata di Sistem (W)

Waktu total pelanggan berada di tempat layanan.

Rumus: W = 1 ÷ (μ - (λ ÷ 2))

Substitusi: W = 1 ÷ ({{ results.mu }} - ({{ results.lam }} ÷ 2))
Proses: W = 1 ÷ ({{ results.mu }} - {{ (results.lam / 2)|round(4) }})
Proses: W = 1 ÷ {{ results.calc_denom }}
Hasil: W = {{ results.w }} Menit
4. Waktu Rata-rata di Antrian (Wq)

Waktu pelanggan menunggu sebelum dipanggil.

Rumus: Wq = λ² ÷ [2μ(μ - λ/2)]

Pembilang (Atas): {{ results.lam }}² = {{ results.lam_sq }}
Penyebut (Bawah): 2({{ results.mu }}) × ({{ results.calc_denom }}) = {{ (results.calc_2mu * results.calc_denom)|round(5) }}
Hitung: Wq = {{ results.lam_sq }} ÷ {{ (results.calc_2mu * results.calc_denom)|round(5) }}
Hasil: Wq = {{ results.wq }} Menit
5. Menghitung Jumlah Orang (Hukum Little)

L (Jumlah orang di sistem): λ × W
{{ results.lam }} × {{ results.w }} = {{ results.l }} Orang

Lq (Jumlah orang antri): λ × Wq
{{ results.lam }} × {{ results.wq }} = {{ results.lq }} Orang

Grafik menunjukkan perbandingan Waktu di Sistem (W) seiring bertambahnya Laju Kedatangan (λ).
Biru: M/M/2 Merah: M/M/1

Panduan Membaca Grafik

1. Interpretasi Sumbu:

  • Sumbu X (λ): Semakin ke kanan, pelanggan datang semakin sering (laju tinggi).
  • Sumbu Y (W): Semakin ke atas, waktu yang dihabiskan pelanggan di sistem semakin lama.

2. Analisis Garis:

  • Garis Biru (M/M/2): Menunjukkan bahwa dengan 2 pelayan, kenaikan waktu tunggu jauh lebih landai dan stabil.
  • Garis Putus Merah (M/M/1): Menunjukkan lonjakan waktu yang sangat tajam saat λ mendekati μ.

Titik Kuning: Posisi sistem Anda saat ini berdasarkan input yang Anda masukkan. Titik ini menunjukkan di mana λ ({{ results.lam }}) memotong kurva M/M/2 pada waktu W ({{ results.w }}).

Catatan: Menambahkan 1 pelayan tambahan (dari M/M/1 ke M/M/2) secara signifikan "menekan" grafik ke bawah, memberikan keamanan extra terhadap antrian panjang yang tak terduga.
